# CHATBOT
## Chatbot1 para Ferretería Yenri
Este proyecto es una aplicación de escritorio construida con Electron y React, diseñada para automatizar el envío de mensajes de WhatsApp a clientes deudores de Ferretería Yenri. Permite gestionar clientes, excepciones, fechas de envío y visualizar resultados de los mensajes enviados.

## Características
Envío automático de mensajes personalizados a clientes con deudas.
Gestión de clientes y lista de excepciones desde archivos Excel.
Visualización de clientes, deudores y resultados de envío.
Copia rápida de nombres y datos de clientes fallidos.
Integración con WhatsApp Web usando puppeteer y whatsapp-web.js.
Interfaz moderna y fácil de usar.
## Instalación
Clona el repositorio.
Instala las dependencias con npm install.
Ejecuta la aplicación con npm run dev.
Estructura del proyecto
main: Lógica principal de Electron y backend.
renderer: Interfaz de usuario en React.
excel: Archivos Excel para excepciones y clientes.
resources: Recursos gráficos y de configuración.
## Uso
Inicia sesión en WhatsApp Web desde la app.
Visualiza y gestiona clientes y excepciones.
Envía mensajes automáticos a los deudores.
Consulta los resultados y copia datos fácilmente.
## Requisitos
Node.js
Google Chrome (para puppeteer)
WhatsApp Web
